## Getting Started

```
 cd backend
 npm install
 nvm use 20 (otherwise OpenCV will not work)
 npm run build && npm start
```

 
and in a new terminal:
```
 cd video_frontend
 npm install
 npm run dev
 npx react-native run-android
```

http://localhost:5001 should run your server now, and the emulator should pop up.
